    Collect a file by copying from a running docker container

  • docker.container-exec

    Collect the stdout and stderr of `exec`-ing a command on a running docker container

  • docker.container-inspect

    Collect the `docker inspect` output for one or more running or stopped containers. One of `container` or `container_list_options` is required.

  • docker.container-logs

    Collect the stdout/stderr logs from one or more docker containers. One of `container` or `container_list_options` is required.

  • docker.container-ls

    Collect information about one or more containers

  • docker.container-run

    Collect the stderr/stdout of running a single docker container

  • docker.exec

    Collect the stdout/stderr of executing a command in an already running docker container

  • docker.image-ls

    Collect a list of docker images present on the server

  • docker.images

    Collect a list of docker images present on the server

  • docker.info

    Collect info about the Docker daemon

  • docker.logs

    Collect the stdout/stderr logs from one or more docker containers. One of `container` or `container_list_options` is required.

  • docker.node-ls

    Collect information about the nodes in a Docker Swarm installation

  • docker.ps

    Collect information about containers

  • docker.run

    Collect the stderr/stdout of running a single docker container

  • docker.service-logs

    Collect logs from a docker swarm service. One of `service` or `service_list_options` is required.

  • docker.service-ls

    Collect a list of docker swarm services

  • docker.service-ps

    Collect information about the tasks run by one or more services

  • docker.stack-service-logs

    Collect logs from one or more services in a stack

  • docker.stack-service-ls

    Collect information about services in a swarm stack

  • docker.stack-service-ps

    Collect information about the tasks running in a service

  • docker.stack-task-logs

    Collect logs from a docker swarm task

  • docker.task-logs

    Collect logs from a docker swarm task

  • docker.task-ls

    List docker swarm tasks

  • docker.version

    Get the version of the docker server

  • journald.logs

    Collect logs from journald

  • os.hostname

    Get the hostname

  • os.http-request

    Collect information about the response from making an HTTP request

  • os.loadavg

    Collect information about CPU load

  • os.read-file

    Read a file from the filesystem

  • os.run-command

    Run a command. Will be run in the support bundle container if `docker run replicated/support-bundle` is used. To run in a custom container, use `docker.run` instead

    Collect information about the system's uptime
